# ModelCraft 图标生成 Prompt 清单（P0–P2）

---

## 通用风格前缀（每条 prompt 必须包含）

```
flat vector icon, minimal line-art, stroke weight 1.5px, monochrome #4F46E5 indigo,
transparent background, 24x24 pixel grid, enterprise SaaS style similar to Stripe dashboard,
clean geometric shapes, no gradients, no shadows, no decorative elements, no text
```

---

## P0 — 品牌图标（4 个）

> 优先级最高，直接影响产品识别度。建议单独生成，反复迭代到满意再批量生产其他图标。

---

**用途**：登录页 Logo、侧边栏组织徽标、AI 功能标记

```

clean geometric shapes, no gradients, no shadows, no decorative elements, no text,
a four-pointed sparkle star: one large 4-pointed star in center, two smaller 4-pointed stars
at upper-right and lower-left, all stars have sharp diamond points
```
