## 页面信息

- **页面名称**: 字段编辑页（Format 不可变）
- **页面类型**: 编辑页（更新）
- **路由路径**: 待定（沿用现有字段编辑路由）
- **所属领域**: 模型设计 / 字段管理

## 数据依赖

| 领域实体 | 使用方式 | 关键字段 |
|----------|----------|----------|
| Field | 加载存量字段并提交更新 | id, name, description, format, relateEnumName, enumRelationId |
| Enum | 仅用于展示已绑定枚举信息（ENUM） | name |
| FieldEnumRelation | 仅用于展示已绑定关系（ENUM_LABEL） | id |
| FieldUpdatePayload | 更新请求载荷 | 非 `format` 字段 |

## 区块说明

### 1. 基本信息区（可编辑）
- **用途**: 维护可变属性。
- **包含字段**: `name`、`description` 等非 `format` 字段。
- **交互**: 正常编辑并参与保存。

### 2. 格式信息区（只读）
- **用途**: 明确 `format` 固化状态，避免误操作。
- **包含字段**:
  - `format`
  - 当 `format=ENUM`：只读展示 `relateEnumName`
  - 当 `format=ENUM_LABEL`：只读展示 `enumRelationId`
- **交互**: 不提供输入控件；仅展示不可变提示。

### 3. 操作区
- **用途**: 执行取消或保存。
- **包含字段**: 无。
- **交互**: 保存时禁止携带 `format` 及关联字段变更。

## 用户操作

| 操作 | 触发方式 | 影响 |
|------|----------|------|
| 修改名称/描述 | 编辑输入框 | 允许保存 |
| 试图修改 `format` | 通过页面操作 | 页面无入口，不允许修改 |
| 试图修改 `relateEnumName`（ENUM） | 通过页面操作 | 页面无入口，不允许修改 |
| 试图修改 `enumRelationId`（ENUM_LABEL） | 通过页面操作 | 页面无入口，不允许修改 |
| 绕过前端篡改载荷 | 直接调用接口 | 后端拒绝并返回 `FIELD_FORMAT_IMMUTABLE` |
| 点击保存 | 点击 `[保存]` | 仅提交非 `format` 字段 |

## 错误状态

| 错误场景 | 提示内容 | 来源 |
|----------|----------|------|
| 请求中 `format` 与存量不一致 | 字段格式创建后不可修改 | 后端 `FIELD_FORMAT_IMMUTABLE` |
| 请求中 `relateEnumName` 与存量不一致（ENUM） | 字段格式创建后不可修改 | 后端 `FIELD_FORMAT_IMMUTABLE` |
| 请求中 `enumRelationId` 与存量不一致（ENUM_LABEL） | 字段格式创建后不可修改 | 后端 `FIELD_FORMAT_IMMUTABLE` |
| 提交时关联对象已失效/删除（按字段类型） | 关联配置不存在或不可用 | 后端 `ENUM_NOT_FOUND` / `FIELD_ENUM_RELATION_NOT_FOUND` |
| 保存失败（通用） | 保存失败，请稍后重试 | 前端通用错误兜底 |

## 设计决策

1. 将 `format` 在编辑页固定为只读展示，消除“可改”误导。  
2. 按字段类型展示对应关联信息：`ENUM -> relateEnumName`，`ENUM_LABEL -> enumRelationId`。  
3. 前端不暴露修改入口，后端按存量对比请求做不可变校验，防止绕过。  

## 不做什么

- 不提供编辑页内的“切换字段类型”迁移能力。
- 不支持在编辑页改绑枚举或改绑 relation。
- 不包含批量编辑字段格式的能力。
