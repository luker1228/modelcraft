## 页面信息

- **页面名称**: 租户登录页
- **页面类型**: 入口页（无需 Token）
- **路由路径**: `/tenant/login`
- **所属领域**: Auth（租户端）

## 数据依赖

| 领域实体 | 使用方式 | 关键字段 |
|----------|----------|----------|
| — | 无业务实体，触发 OAuth 跳转 | — |

## 区块说明

### 1. Logo 区
- **用途**: 品牌识别
- **包含字段**: 产品名称、Logo 图标
- **交互**: 无

### 2. 登录按钮区
- **用途**: 触发 OAuth Provider 跳转
- **包含字段**: 按钮文案
- **交互**: 点击 → BFF 调用 `GET /api/auth/login-url` 获取 OAuth URL → 浏览器跳转

### 3. 底部说明区
- **用途**: 区分入口，防止 EndUser 误入
- **包含字段**: 说明文案："仅限租户管理员 / 开发者使用"
- **交互**: 无（可含跳转用户端链接，v1 不做）

## 用户操作

| 操作 | 触发方式 | 影响 |
|------|----------|------|
| 触发 OAuth 登录 | 点击"使用企业账号登录" | BFF 获取 loginUrl → 跳转 OAuth Provider |
| OAuth 回调成功 | OAuth Provider 重定向 `/auth/callback` | 交换 code → 颁发 `aud=tenant` Token → 跳转 `/org-selector` |
| 已登录访问 | 已有有效 Token 访问 `/tenant/login` | 自动重定向 `/org-selector` |

## 错误状态

| 错误场景 | 提示内容 | 来源 |
|----------|----------|------|
| OAuth 回调失败（code 无效）| "登录失败，请重试" | BFF `/auth/callback` 返回 401 |
| OAuth Provider 不可用 | "登录服务暂时不可用，请稍后重试" | BFF 超时或 502 |

## 设计决策

- `/login` 旧路由废弃，不保留重定向，强制迁移到 `/tenant/login`
- 登录页本身不持有 OAuth 逻辑，只触发跳转；实际 Token 颁发在 `/auth/callback` 完成
- `aud=tenant` Token 由后端在 callback 阶段写入，前端透明

## 不做什么

- 不提供用户名/密码表单（租户端只走 OAuth）
- 不在此页面展示 Org 列表（登录后跳 `/org-selector`）
- 不保留旧 `/login` 重定向
