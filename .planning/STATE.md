## 累积上下文

### 关键决策

决策详情见 PROJECT.md Key Decisions 表。当前影响开发的决策：

- 单一 issuer `mc-platform`：消除中间件分叉，一条验证路径
- `scope` claim：向后兼容的 JWT 扩展，无需新签名密钥
- Project Token 不含 `projectSlug`：RBAC 是访问控制面，而非 token payload
- 硬切旧 issuer：1h TTL，用户在一小时内自然重新认证
- httpOnly cookie 存储 Project Token：BFF 负责 exchange，JS 不可读
