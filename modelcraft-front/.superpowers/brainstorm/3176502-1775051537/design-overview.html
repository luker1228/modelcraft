④ 关键决策说明

JSON Schema 的不足如何补？
JSON Schema 缺少 ENUM 选项列表和 RELATION 目标模型信息。解决方案：buildUiSchema() 从 GET_MODEL fields[] 提取这些信息，注入到 uiSchema 的 ui:options 里，传给对应 widget。

RJSF 主题？
使用 @rjsf/shadcn（shadcn 主题），与项目现有 UI 组件库保持一致，无需额外样式适配。

RelationPicker 怎么实现？
作为 RJSF custom widget，内部调用 buildFindManyQuery() 查询关联模型的记录，显示为可搜索下拉框，选中后存储主键值。
