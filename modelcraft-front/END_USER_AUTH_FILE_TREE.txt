DATA FLOW DIAGRAM
═════════════════════════════════════════════════════════════════════════════════════════════

LOGIN
─────
[Browser]
   │ POST /api/bff/org/{orgName}/end-user/auth/login
   ↓
[BFF refresh/route.ts] ────→ [Go Backend /api/end-user/auth/login]
   ↓                              ↓
   ← accessToken, expiresAt        ← refreshToken (Set-Cookie)
   ↓
[Browser]
   │ Store in useEndUserAuthStore:
   │  • accessToken (in-memory)
   │  • expiresAt (in-memory)
   │  • mc_enduser_refresh_token (HttpOnly cookie)
   ↓
[Ready to use]


GRAPHQL QUERY (Apollo)
──────────────────────
[React Component]
   │ useLazyQuery() / useQuery()
   ↓
[Apollo Link] ──────────────────────────────┐
   │ Get token: getEndUserToken()           │
   │ Check: isTokenExpired()? [5 min buffer]│  
   │ If YES: refreshEndUserAccessToken()   │
   │         (with org/projectSlug params)  │
   │ ← Update store + cookie rotation       │
   │ Build headers:                         │
   │  • Authorization: Bearer {token}      │
   │  • X-Action: "query:RecordSearch"     │
   │  • X-Client-Request-Id: UUID          │
   ↓
[BFF /api/bff/graphql/end-user/org/{org}/project/{prj}]
   │
   ↓
[Gateway] ─ Validates JWT ─ Injects X-User-Type: end_user
   │
   ↓
[Go Backend] ─ Enforces end-user permissions
   │
   ↓ Returns GraphQL result
[Apollo Cache] ─ Store result
   │
   ↓
[React Component] ─ Display data


REFRESH (Silent)
────────────────
[Timer / Apollo Link Check]
   │ isTokenExpired()? → YES (now > expiresAt - 5min)
   ↓
[refreshEndUserAccessToken()]
   │ Concurrent guard: _isRefreshing && _refreshPromise?
   │ → YES: return shared promise
   │ → NO: proceed
   ↓
[BFF POST /api/bff/org/{orgName}/end-user/auth/refresh]
   ├─ req.cookies.get(mc_enduser_refresh_token) → extracts refreshToken
   ├─ injects to body: { orgName, refreshToken }
   ↓
[Go Backend /api/end-user/auth/refresh]
   ├─ Validates refresh token
   ├─ Rotates refresh token (new cookie)
   ├─ Issues new access token (JWT)
   ↓ Response: { accessToken, expiresAt, Set-Cookie }
[BFF Response]
   ├─ Transparently pass Set-Cookie to browser
   ├─ Return EndUserAuthResponse
   ↓
[useEndUserAuthStore]
   ├─ setAccessToken(newToken, expiresIn)
   ├─ Decode JWT: extract org_name, project_slug, sub
   ├─ setUserInfo() auto-populated
   ↓
[Ready to use]


LOGOUT
──────
[User Click]
   ↓
[BFF POST /api/bff/org/{orgName}/end-user/auth/logout]
   │ (credentials: include) ← sends HttpOnly cookie
   ↓
[Go Backend /api/end-user/auth/logout]
   │ Revoke refresh token
   ↓ OK
[BFF Response]
   ├─ + Set-Cookie: mc_enduser_refresh_token=; Max-Age=0; HttpOnly
   ↓
[Browser]
   ├─ Cookie cleared
   ├─ useEndUserAuthStore.clearSession()
   │  • accessToken = null
   │  • expiresAt = null
   │  • userInfo = null
   ├─ Apollo cache cleared
   ↓
[Redirect] → /end-user/{orgName}/login
